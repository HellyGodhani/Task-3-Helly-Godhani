# 🌱 Project 3 — The Visual Identity
### UI/UX Design Internship | DecodeLabs 

---

## 📌 Project Overview

This project is part of the DecodeLabs UI/UX Design Internship (DI03000061).
The goal was to create a High-Fidelity Visual Identity for a premium Organic Food Delivery app called SoilToSoul — communicating trust, health, and freshness through colors, typography, and design.

---

## 🎯 Objective

Transform low-fidelity wireframes into a premium High-Fidelity UI Design with a complete Visual Identity.

---

## 🛠️ Tool Used
Figma — UI/UX Design Tool

---

## 🎨 Brand Identity — SoilToSoul

Tagline: Pure. Fresh. Yours.

Color Palette:
- Leaf Green #4A7C59 — Buttons, Headings
- Earthy Brown #5C3D2E — Text, Labels
- Harvest Gold #C9A84C — Accent moments
- Cream White #F5F0E8 — Background
- Pure White #FFFFFF — Cards, Surfaces

Typography:
- Playfair Display (Serif) — Headlines, Product Names
- Montserrat (Sans-Serif) — Body Text, Prices, Buttons
